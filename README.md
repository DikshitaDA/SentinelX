# 🚨 SentinelX – Intelligent Security Monitoring Platform

SentinelX is a lightweight Security Monitoring System inspired by enterprise SIEM tools.  
It is designed to ingest logs, detect suspicious activity, and provide basic threat monitoring capabilities.

---

## 📌 Project Objective

The goal of SentinelX is to:

- Understand how security monitoring systems work
- Build a log ingestion pipeline
- Implement rule-based detection logic
- Store logs securely in a database
- Visualize suspicious activities

This project is built as part of a hands-on cybersecurity learning journey.

## 🔧 Setup Instructions

1. Clone the repository  
2. Create virtual environment:
   python -m venv venv  

3. Activate environment (Git Bash):
   source venv/Scripts/activate  

4. Install dependencies:
   pip install -r requirements.txt  

5. Run application:
   python app.py  

6. Open browser:
   http://127.0.0.1:5000
